CSS Techniques 07 / 08

CSS
Glassmorphism

backdrop-filter:blur() blurs everything behind an element. Combined with translucent backgrounds and subtle borders, it creates frosted glass UI.

Type 01

Frosted Glass Panel

The classic recipe: a colourful background behind, then a glass layer with backdrop-filter:blur(), background:rgba(255,255,255,.1), and a thin white border.

Frosted Glass Layer
.glass { background: rgba(255, 255, 255, 0.1); backdrop-filter: blur(20px); -webkit-backdrop-filter: blur(20px); border: 1px solid rgba(255, 255, 255, 0.2); border-radius: 12px; }

Type 02

Dark Glass

On dark backgrounds use a low-opacity dark background instead of white. A stronger blur radius and gradient border reinforce depth. An inset box-shadow adds a recessed look.

Dark Glass
.dark-glass { background: rgba(0, 0, 0, 0.4); backdrop-filter: blur(16px); border: 1px solid rgba(255,255,255,.08); box-shadow: inset 0 1px 0 rgba(255,255,255,.05); }

Type 03

Layered Depth

Stack multiple glass panels at different blur levels to simulate depth. Each layer blurs progressively more. The result is parallax-style depth without any JS transform calculation.

Deepest Layer
/* Progressive blur depth */ .layer-1 { backdrop-filter: blur(4px); background: rgba(255,255,255,.04); } .layer-2 { backdrop-filter: blur(10px); background: rgba(255,255,255,.07); } .layer-3 { backdrop-filter: blur(20px); background: rgba(255,255,255,.10); }

Type 04

Saturate & Brightness

Combine filter functions: blur() + saturate() + brightness(). macOS and iOS UIs oversaturate to boost colour pop behind frosted glass. Replicate it entirely in CSS.

Saturated Glass
.macos-glass { backdrop-filter: blur(8px) saturate(180%) brightness(1.1); background: rgba(255, 255, 255, 0.35); /* Multiple filter functions = space-separated */ }
