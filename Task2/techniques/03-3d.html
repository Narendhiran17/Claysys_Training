3D Transforms

Pure CSS 3D uses perspective, transform-style:preserve-3d, and 3D transform functions to build spatial depth without WebGL or canvas.

Type 01

Rotating Cube

Six faces positioned with translateZ and rotateY/X. The parent needs transform-style:preserve-3d so faces render in real 3D space, not flat.

.wrap { perspective: 400px; } .cube { transform-style: preserve-3d; animation: rotateCube 6s linear infinite; } @keyframes rotateCube { from { transform: rotateX(0) rotateY(0); } to { transform: rotateX(360deg) rotateY(360deg); } } .face-front { transform: translateZ(30px); } .face-back { transform: rotateY(180deg) translateZ(30px); } .face-left { transform: rotateY(-90deg) translateZ(30px); }

Type 02

Card Flip

Outer wrapper holds perspective, inner wrapper holds preserve-3d and transitions rotateY. The back face is pre-rotated 180° and hidden with backface-visibility:hidden.

FRONT
BACK

hover card →

.scene { perspective: 600px; } .card { transform-style: preserve-3d; transition: transform .6s; } .card:hover { transform: rotateY(180deg); } .front, .back { backface-visibility: hidden; } .back { transform: rotateY(180deg); }

Type 03

Hover Tilt

Combine rotateY, rotateX, and translateZ on hover for a parallax tilt. The spring easing cubic-bezier(.34,1.56,.64,1) gives a physical feel.

TILT

hover →

.scene { perspective: 600px; } .card { transition: transform .6s cubic-bezier(.34,1.56,.64,1); } .card:hover { transform: rotateY(25deg) rotateX(-10deg) translateZ(20px); }

Type 04

Coin Flip

A single element animated with rotateY — as it passes 90° the shape appears to vanish and re-emerge. Combine with a radial gradient for a metallic sheen.

@keyframes flipCoin { 0% { transform: rotateY(0); } 50% { transform: rotateY(90deg); } 100% { transform: rotateY(180deg); } } .coin { animation: flipCoin 3s ease-in-out infinite; }
