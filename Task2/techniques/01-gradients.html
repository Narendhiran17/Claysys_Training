CSS Techniques 01 / 08

Conic &
Gradients

Pure CSS colour transitions — linear, radial, conic, and repeating variants for backgrounds, borders, and text without a single image.

Type 01

Linear Gradient

Transitions between colours along a straight line. The angle controls direction — to right, 135deg, or any value. Multi-stop gradients produce rich results.

.element { background: linear-gradient( 135deg, #c8ff00 0%, #00f0d4 50%, #ff3f6c 100% ); }

Type 02

Radial Gradient

Colours radiate from a centre point. The at keyword positions the focal point. Layer multiple radial gradients for orb / light leak effects.

.element { background: radial-gradient(ellipse at 20% 80%, #00f0d4 0%, transparent 50%), radial-gradient(ellipse at 80% 20%, #ff3f6c 0%, transparent 50%), #050508; }

Type 03

Conic Gradient

Colours rotate around a centre point — like a colour wheel. The from parameter sets the start angle. Pair with hue-rotate animation for an infinite colour spin.

@keyframes hueRotate { to { filter: hue-rotate(360deg); } } .element { background: conic-gradient( from 180deg at 50% 70%, #c8ff00 0deg, #00f0d4 90deg, #ff3f6c 180deg, #f5c842 270deg, #c8ff00 360deg ); animation: hueRotate 4s linear infinite; }

Type 04

Repeating Gradient

Tiles a gradient pattern endlessly. Works with all gradient types. repeating-linear-gradient at 45° creates diagonal stripes with no images.

.element { background: repeating-linear-gradient( 45deg, #050508 0px, #050508 10px, #1a1a2e 10px, #1a1a2e 20px ); }

Type 05

Animated Gradient Shift

Animate a gradient by shifting background-position on an oversized background-size. CSS can't tween gradient colours directly — this moves a 300%-wide gradient left-to-right to simulate colour flow for text, borders, and backgrounds.

@keyframes gradShift { 0%,100% { background-position: 0% center; } 50% { background-position: 100% center; } } .element { background: linear-gradient(90deg, #c8ff00,#00f0d4,#ff3f6c,#f5c842,#c8ff00); background-size: 300% auto; animation: gradShift 5s ease infinite; }
