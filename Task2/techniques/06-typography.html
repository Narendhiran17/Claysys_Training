CSS Techniques 06 / 08

CSS
Typography

Beyond font choice — CSS gives you kinetic animation, fluid sizing with clamp(), typewriter effects, and rotated text flow, all without JavaScript.

Type 01

Kinetic Expansion

Animate letter-spacing to make text breathe. Go from 0em to .3em with ease-in-out infinite alternate. Combine with gradient text for a premium kinetic headline.

EXPAND
@keyframes stretchText { from { letter-spacing: 0em; } to { letter-spacing: .3em; } } .kinetic { animation: stretchText 3s ease-in-out infinite alternate; }

Type 02

Fluid Type with clamp()

clamp(min, preferred, max) scales font sizes fluidly. The preferred value uses viewport units like 5vw. No media queries needed — text resizes smoothly between any screen widths.

SMALL
MED
LARGE
h1 { font-size: clamp(40px, 8vw, 120px); /* min | preferred | max */ /* Resizes between 40px and 120px based on viewport width — no @media */ }

Type 03

Typewriter Effect

steps() timing makes animation jump in discrete increments. Animate width from 0 to 100% on a monospaced overflow-hidden element, then add a blinking border-right cursor.

Hello, World_
@keyframes typewriter { from { width: 0; } to { width: 100%; } } @keyframes blinkCursor { 50% { border-color: transparent; } } .text { overflow: hidden; white-space: nowrap; border-right: 2px solid currentColor; animation: typewriter 3s steps(20) infinite alternate, blinkCursor .6s step-end infinite; }

Type 04

writing-mode & Vertical Text

writing-mode:vertical-lr rotates text flow 90°. Unlike rotate(-90deg), this changes the layout axis — text wraps vertically, block/inline directions swap. Add rotate(180deg) to flip direction.

VERTICAL
TYPE

Block layout follows text flow — not just a visual rotation.

.sidebar-label { writing-mode: vertical-lr; transform: rotate(180deg); /* inline = vertical, block = horizontal unlike rotate() which keeps layout horizontal */ }
