Shape Shift
Clip-Path

clip-path masks an element to any geometric shape. The key power: it is fully animatable — morph between shapes smoothly with CSS transitions.

Type 01

Polygon

Define any custom shape with X,Y coordinate pairs. polygon(50% 0%, 100% 100%, 0% 100%) is a triangle. Points are percentages of the element's bounding box — fully scalable.

/* Triangle */ clip-path: polygon(50% 0%, 100% 100%, 0% 100%); /* Hexagon */ clip-path: polygon( 25% 0%, 75% 0%, 100% 50%, 75% 100%, 25% 100%, 0% 50% ); /* 5-point Star */ clip-path: polygon( 50% 0%,61% 35%,98% 35%,68% 57%, 79% 91%,50% 70%,21% 91%,32% 57%, 2% 35%,39% 35% );

Type 02

Circle & Ellipse

circle(r at x y) clips to a circle of radius r. ellipse(rx ry at x y) gives an oval. Both are animatable — great for reveal transitions on hover.

/* Circle centred */ clip-path: circle(50%); /* Ellipse */ clip-path: ellipse(50% 35% at 50% 50%); /* Hover reveal from centre */ .el { clip-path: circle(0%); transition: .5s; } .el:hover { clip-path: circle(100%); }

Type 03

Animated Morph

Morphing requires the same number of points in both shapes. The browser interpolates each coordinate individually. Hover the demo below.

.shape { clip-path: polygon(50% 0%,100% 100%,0% 100%); transition: clip-path .5s cubic-bezier(.34,1.56,.64,1); } /* Same point count required for morphing */ .parent:hover .shape { clip-path: polygon(0 0,100% 0,100% 100%,0 100%); }

Type 04

Inset & Diagonal Cut

inset() clips a rectangle from each edge with optional round keyword for radius. Diagonal hero sections use a trapezoid polygon — no extra elements or borders needed.

/* Inset with rounded corners */ clip-path: inset(5px 10px round 8px); /* Diagonal / parallelogram */ clip-path: polygon( 0 0, 100% 0, 96% 100%, 4% 100% ); /* Hero diagonal cut */ clip-path: polygon( 0 0, 100% 0, 100% 90%, 0 100% );
