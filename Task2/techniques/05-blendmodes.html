CSS Techniques 05 / 08

CSS
Blend Modes

mix-blend-mode and background-blend-mode composite layers using Photoshop-style blending — all within pure CSS.

Type 01

Multiply

Multiplies RGB values — dark areas stay dark, white becomes transparent. Requires a white background to see the overlap effect correctly.

.circle { mix-blend-mode: multiply; } /* White background needed for multiply */ .container { background: white; }

Type 02

Screen

Inverse of multiply — lightens by inverting, multiplying, then inverting again. Black disappears; overlapping areas get brighter. Works on dark backgrounds for glow / light-leak effects.

.circle { mix-blend-mode: screen; } /* Dark background for screen blend */ .container { background: #050508; }

Type 03

Exclusion

Similar to difference but lower contrast — produces a vivid colour inversion. The custom cursor in this very site uses mix-blend-mode:exclusion on a lime dot to invert whatever colour is underneath.

EXCLUSION
#cursor { background: #c8ff00; mix-blend-mode: exclusion; /* Inverts whatever colour is underneath */ }

Type 04

Overlay & Soft Light

overlay multiplies dark areas and screens light areas — boosts contrast. soft-light is gentler: darkens darks, lightens lights, preserving midtones better.

overlay
soft-light
.overlay-layer { mix-blend-mode: overlay; } .softlight-layer { mix-blend-mode: soft-light; } /* Other useful values: */ /* hard-light, color-dodge, color-burn, luminosity, hue, saturation, color */

Type 05

background-blend-mode

Blends an element's own background layers against each other — not against other elements. Use it to mix a gradient over a background-image in one declaration, keeping them together as a unit.

.element { background: linear-gradient(135deg, rgba(200,255,0,.6), rgba(255,63,108,.6)), url('texture.png'); background-blend-mode: multiply; /* Blends layers against each other, not against elements beneath */ }
