CSS Techniques 02 / 08

Keyframe
Animations

CSS animations choreograph change over time using @keyframes. Control timing, easing, direction, and fill-mode with the animation shorthand.

Type 01

Stagger with Delay

Apply the same @keyframes to multiple elements but offset each with increasing animation-delay. The rhythmic cascade is entirely CSS — no JS.

@keyframes barDance { from { height: 20%; } to { height: 100%; } } .bar { animation: barDance 1.4s ease-in-out infinite alternate; } .bar:nth-child(2) { animation-delay: .15s; } .bar:nth-child(3) { animation-delay: .30s; }

Type 02

Continuous Spin

linear timing with infinite produces a seamless rotation. Animating transform:rotate() is GPU-accelerated — the smoothest CSS animation possible.

@keyframes spin { to { transform: rotate(360deg); } } .element { animation: spin 2s linear infinite; }

Type 03

Scale Pulse

Scaling with transform:scale() is always hardware-accelerated. alternate oscillates back and forth without writing both directions explicitly.

@keyframes pulse { 0%,100% { transform: scale(1); } 50% { transform: scale(1.15); } } .element { animation: pulse 1.5s ease-in-out infinite alternate; }

Type 04

Float & Bounce

Vertical float via translateY — always prefer transform over animating top or margin to avoid layout recalculation. ease-in-out gives a natural arc.

@keyframes bounceY { 0%,100% { transform: translateY(0); } 50% { transform: translateY(-20px); } } .element { animation: bounceY 2s ease-in-out infinite; }

Type 05

Shape Morph

Animating border-radius through percentage values morphs elements between circles, blobs, and rectangles. Each corner is set independently for organic shifting shapes.

@keyframes morphPath { 0%,100% { border-radius: 50%; } 25% { border-radius: 10% 40% 40% 10%; } 50% { border-radius: 0; } 75% { border-radius: 40% 10% 10% 40%; } }
