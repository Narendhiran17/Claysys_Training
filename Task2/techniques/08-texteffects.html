CSS Techniques 08 / 08

CSS
Text Effects

From gradient fills and stroke outlines to neon glows and glitch splits — CSS gives you full control over how text appears, without touching a raster image.

Type 01

Gradient Text

Set a gradient as background, then clip it to the text shape with -webkit-background-clip:text and make the text colour transparent with -webkit-text-fill-color:transparent.

GRADIENT
DUAL TONE
.gradient-text { background: linear-gradient(90deg, #ff3f6c, #f5c842, #c8ff00, #00f0d4); background-size: 300% auto; -webkit-background-clip: text; -webkit-text-fill-color: transparent; background-clip: text; }

Type 02

Text Stroke / Outline

-webkit-text-stroke adds a stroke around each glyph. Set -webkit-text-fill-color:transparent for hollow outlines. Unlike text-shadow, stroke renders inside the letterform.

OUTLINE
STROKE
.stroke-text { -webkit-text-stroke: 1.5px #00f0d4; -webkit-text-fill-color: transparent; letter-spacing: .05em; transition: letter-spacing .5s; } .stroke-text:hover { letter-spacing: .3em; }

Type 03

Neon Glow

Layer multiple text-shadow values at increasing blur radii. Animate opacity or shadow size between two states to pulse. No image, no filter — pure CSS property stacking.

NEON
ROUGE
@keyframes neonPulse { from { text-shadow: 0 0 5px #c8ff00, 0 0 15px #c8ff00; } to { text-shadow: 0 0 10px #c8ff00, 0 0 40px #c8ff00, 0 0 80px #c8ff00; } }

Type 04

Glitch Effect

Duplicate text using ::before / ::after with content:attr(data-text), colour each a different accent, then animate clip-path and translate to slice and shift them.

GLITCH
.glitch::before, .glitch::after { content: attr(data-text); position: absolute; top: 0; left: 0; } .glitch::before { color: #ff3f6c; animation: glitchShift 3s infinite; } .glitch::after { color: #00f0d4; animation: glitchShift 3s .1s infinite; } @keyframes glitchShift { 20% { clip-path:inset(30% 0 50% 0); transform:translate(-4px,2px); } 60% { clip-path:inset(10% 0 80% 0); transform:translate(4px,-2px); } }
